# Habit Tracker App

This is a **Habit Tracker App** built with **React + Vite**.  
The app allows users to track daily habits, set goals, and monitor their progress.

## Features (Work in Progress)
- Add, edit, and delete habits
- Track daily progress
- Set weekly goals
- Dashboard to visualize habits

## Tech Stack
- React + Vite for frontend
- JavaScript for core logic
- (Future) Node.js / backend integration
